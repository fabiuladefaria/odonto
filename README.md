# Clínica Odontológica – Site Demonstrativo

Este projeto foi desenvolvido exclusivamente para fins de demonstração e apresentação de portfólio.

O objetivo é ilustrar como pode ser um site moderno, responsivo e profissional para clínicas odontológicas, servindo como exemplo de layout, design e experiência do usuário.

As informações, imagens, nomes, depoimentos, equipe, endereço e demais conteúdos apresentados são fictícios e utilizados apenas para demonstração.

Este projeto **não representa uma clínica real**, não possui vínculo com nenhuma empresa ou profissional da área da saúde e não deve ser utilizado como site oficial sem as devidas adaptações.

## Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* Bootstrap 5
* AOS (Animate On Scroll)
* Font Awesome
* Google Fonts

## Objetivo

Este projeto faz parte do portfólio da **Faria Tec** e demonstra a capacidade de desenvolver sites institucionais modernos, rápidos, responsivos e otimizados para empresas e profissionais.
